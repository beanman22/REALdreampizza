

*to change pizza prices go into, bin > debug > pizzaprices.txt
*to change pizza names go into, bin > debug > pizzanames.txt


*pizzas are listed in decending order, the first seven being standard, and the last five being gourmet.

*each line in pizzaprices.txt correspondes to each line in pizzanames.txt

*for example line one on pizzanames.txt, would have the price of line one in Pizzaprices.txt
